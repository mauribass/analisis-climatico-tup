## Descripción del Dataset
Dataset de temperatura promedio global (GISTEMP) obtenido de datahub.io.
Contiene registros mensuales y anuales de temperatura global en formato CSV.

## Estructura del Repositorio
- /datos: Dataset climático en formato CSV
- /scripts: Script Python de análisis
- /resultados: Gráficos y resultados generados

## Instrucciones para Ejecutar
1. Clonar el repositorio
2. Abrir scripts/analisis_datos.py en Google Colab
3. Ejecutar todas las celdas en orden
4. Los resultados se guardan en /resultados
